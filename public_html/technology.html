HTML

-Hypertext Markup Language

HTML is abbreviated for Hypertext Markup Language. A markup language is a set of mark up tags. HTML documents are described by HTML tags. Each HTML tag describes different document content.

CSS

-Cascading Style Sheet

CSS is abbreviated for Cascading Style Sheet. Styles define how to display HTML elements. External Style Sheets can save a lot of work. External Style sheets are stored in CSS files.
